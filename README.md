projeto_jogo_OO
===============
Jogo do Morro

jogo de plataforma feito em Java.

Regras

Objetivo: O personagem é um policial que sobe o morro atraz de acabar como crime.

Como jogar: Utilize as setas esquerda e direita e a tecla espaço.
